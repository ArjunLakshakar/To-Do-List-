# React To-Do App

A simple and clean To-Do list application built with React and Tailwind CSS.  
You can add, edit, mark complete, and delete tasks.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Clean UI with Tailwind CSS
